顶层容器：JFrame（能独立运行）
	通过方法设置内容窗格：
		1. Container c = frame.getContentPane();
			c.add()
		2. Jpanel cp= new JPanel();//创建面板。就像白纸
			cp.setLayout();
			cp.add();
			frame.setContentPane(cp)
不能独立运行的容器： 普通面板 Jpanel 和 滚动面板 JscrollPane
垂直滚动： void setVerticalScrollBarPolicy(int policy);
水平滚动： void setHorizontalScrollBarPolicy(int policy)
policy的值：
	需要时可见：ScrollPaneConstants.VERTICAL_SCROLLBAR_AS_NEEDED;
	不可见：ScrollPaneConstants.VERTICAL_SCROLLBAR_AS_NEVER;
	可见：ScrollPaneConstants.VERTICAL_SCROLLBAR_AS_ALWAYS;
更小一级滚动条： Scrollbar(int orientation);构造一个具有新方向的滚动条

		Scrollbar(int orientation，int value(大小)，int visible（可视大小），int minimum，int maximum);构造一个新的滚动条
